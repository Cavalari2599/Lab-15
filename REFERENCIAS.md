referencias

estas son las cosas que consulte para hacer el laboratorio.

las presentaciones del curso de multimedios, la de node y la de express, de ahi saque casi toda la estructura del proyecto

https://multimediossg.github.io/slides/nodejs/
https://multimediossg.github.io/slides/express/

la documentacion de node para el modulo de sqlite que se llama node:sqlite, para usar DatabaseSync, las consultas preparadas y las sentencias INSERT, UPDATE y DELETE

https://nodejs.org/api/sqlite.html

la documentacion de node para leer y escribir archivos con node:fs

https://nodejs.org/api/fs.html

la pagina de express para las rutas, los parametros, el middleware express.json y los archivos estaticos

https://expressjs.com/en/guide/routing.html
https://expressjs.com/en/api.html

la pagina de zod para validar el cuerpo de los albumes y el texto de busqueda

https://zod.dev/

para normalizar el titulo y generar el slug use String.prototype.normalize de mdn

https://developer.mozilla.org/es/docs/Web/JavaScript/Reference/Global_Objects/String/normalize

para los datos de cada album como el artista, el genero, el anio y el sello use wikipedia

https://es.wikipedia.org/wiki/Anexo:Albumes_mas_vendidos

las portadas son las imagenes de cada album, las baje de wikipedia usando su api rest con el script downloadImages.js, es solo para fines del laboratorio

https://en.wikipedia.org/api/rest_v1/

tambien use curl para probar la api desde la terminal

https://curl.se/

para la suite de pruebas automatizadas use vitest y supertest

https://vitest.dev/
https://github.com/ladjs/supertest
